Intro

Installable via npm install --save peer-id, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save peer-id

You can then require() peer-id as normal:

const peerId = require('peer-id')

In the Browser

Peer-id should work in any ES2015 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of peer-id, including index.js and index.min.js, are included in the /dist folder. Peer-id can also be found on unpkg.com under

  • https://unpkg.com/peer-id/dist/index.min.js
  • https://unpkg.com/peer-id/dist/index.js
